#  Detnerat — Static Web Project

![Static Badge](https://img.shields.io/badge/Status-Complete-brightgreen?style=flat-square)
![GitHub Pages](https://img.shields.io/badge/Deployment-GitHub_Pages-blue?style=flat-square&logo=github)

Концептуальный сайт коворкинг-кофейни для разработчиков. Проект выполнен с использованием чистого стека технологий без лишних библиотек.

## 🛠 Технологии
* **HTML5**: Семантическая структура.
* **CSS3**: Flexbox, Grid, CSS-переменные для тем.
* **JavaScript**: Управление состоянием (DarkMode), работа с `localStorage`.

## 📁 Страницы проекта
1.  **Главная**: Оффер и ключевые преимущества.
2.  **О нас**: История Detnerat.
3.  **Блог**: Список новостей.
4.  **Контакты**: Контактные данные и интерактивная форма.

## 🎨 Особенности
* **Adaptive Design**: Сайт полностью адаптирован под мобильные устройства.
* **Dark/Light Mode**: Переключатель тем с сохранением выбора пользователя в браузере.
* **No Dependencies**: Только чистый код, максимально быстрая загрузка.

## 🚀 Как запустить
Просто откройте `index.html` в любом браузере или загрузите папку в репозиторий GitHub для автоматического деплоя через **GitHub Pages**.
